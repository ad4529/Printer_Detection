Usage of the annotation tool:

cd into /home/abhisek/Desktop/YoloImages/Yolo-Annotation-Tool-New-

Run main.py ($python3 main.py)

The annotation tool window should start up.

Specify your new Image folder to be annotated in 'Image Dir' field (Make sure your new folder with images is inside /home/abhisek/Desktop/YoloImages/Yolo-Annotation-Tool-New-/Images directory. All the images have a .jpg extension. No .jpeg or .png).
